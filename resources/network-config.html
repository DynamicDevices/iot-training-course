ESP32 Configuration Examples

Static IP Configuration

// Network settings for ESP32
IPAddress local_IP(192, 168, 1, 100);    // Device IP
IPAddress gateway(192, 168, 1, 1);       // Router IP
IPAddress subnet(255, 255, 255, 0);      // Subnet mask
IPAddress primaryDNS(8, 8, 8, 8);        // Primary DNS
IPAddress secondaryDNS(8, 8, 4, 4);      // Secondary DNS

Alternative Network Ranges

Corporate/Enterprise Networks:

  • 10.0.0.0/8 (10.0.0.1 - 10.255.255.254)
  • 172.16.0.0/12 (172.16.0.1 - 172.31.255.254)

Common Router Defaults:

  • Linksys: 192.168.1.1
  • Netgear: 192.168.1.1 or 192.168.0.1
  • TP-Link: 192.168.0.1
  • ASUS: 192.168.1.1

Wi-Fi Security Settings

Recommended Settings:

  • Security: WPA2-PSK or WPA3-PSK
  • Encryption: AES
  • Channel: 1, 6, or 11 (2.4GHz)
  • Channel Width: 20MHz for IoT devices

Avoid These Settings:

  • WEP encryption (deprecated)
  • Open networks (no security)
  • Hidden SSID (causes connection issues)
