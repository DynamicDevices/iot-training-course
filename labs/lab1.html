Lab 1: Network Fundamentals

Hands-on exploration of TCP/IP networking concepts

Lab Overview

Duration: 2 hours Difficulty: Beginner

This lab introduces fundamental networking concepts through practical exercises using command-line tools and network analysis software.

Learning Objectives

  • Understand IP addressing and subnetting
  • Explore DHCP vs static IP configuration
  • Analyze network traffic with basic tools
  • Practice essential network troubleshooting commands
