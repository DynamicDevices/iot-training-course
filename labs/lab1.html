Lab Exercises

Exercise 1: DHCP Configuration

30 minutes

Task:

Configure ESP32 to connect to Wi-Fi using DHCP and display network information.

Starter Code:

#include <WiFi.h>

const char* ssid = "YOUR_WIFI_SSID";
const char* password = "YOUR_WIFI_PASSWORD";

void setup() {
    Serial.begin(115200);
    
    // TODO: Initialize Wi-Fi connection
    WiFi.begin(ssid, password);
    
    // TODO: Wait for connection and display status
    while (WiFi.status() != WL_CONNECTED) {
        delay(1000);
        Serial.println("Connecting to WiFi...");
    }
    
    // TODO: Display network information
    Serial.println("WiFi connected!");
    Serial.print("IP address: ");
    Serial.println(WiFi.localIP());
    Serial.print("Subnet mask: ");
    Serial.println(WiFi.subnetMask());
    Serial.print("Gateway: ");
    Serial.println(WiFi.gatewayIP());
    Serial.print("DNS: ");
    Serial.println(WiFi.dnsIP());
}

void loop() {
    // TODO: Monitor connection status
    if (WiFi.status() == WL_CONNECTED) {
        Serial.println("Connection active");
    } else {
        Serial.println("Connection lost - attempting reconnect");
        WiFi.reconnect();
    }
    delay(10000);
}

Complete these tasks:

  1. Fill in your Wi-Fi credentials
  2. Add error handling for connection failures
  3. Display MAC address and signal strength
  4. Implement automatic reconnection logic

Exercise 2: Static IP Configuration

45 minutes

Task:

Configure ESP32 with a static IP address and compare behavior with DHCP.

Static IP Template:

// Static IP configuration
IPAddress local_IP(192, 168, 1, 184);
IPAddress gateway(192, 168, 1, 1);
IPAddress subnet(255, 255, 255, 0);
IPAddress primaryDNS(8, 8, 8, 8);
IPAddress secondaryDNS(8, 8, 4, 4);

void setup() {
    Serial.begin(115200);
    
    // Configure static IP before connecting
    if (!WiFi.config(local_IP, gateway, subnet, primaryDNS, secondaryDNS)) {
        Serial.println("Static IP configuration failed");
    }
    
    WiFi.begin(ssid, password);
    
    // TODO: Add connection monitoring and comparison
}

Implementation Requirements:

  1. Create a function to switch between DHCP and static IP
  2. Measure and compare connection times
  3. Test network connectivity with both configurations
  4. Document the differences in behavior

Exercise 3: Network Monitoring & Diagnostics

45 minutes

Task:

Implement comprehensive network monitoring and diagnostic functions.

Features to Implement:

  1. Connection Health Monitor: Track connection stability
  2. Network Scanner: Discover other devices on network
  3. Ping Function: Test connectivity to specific hosts
  4. Speed Test: Measure network throughput

Diagnostic Functions Template:

void networkDiagnostics() {
    Serial.println("=== Network Diagnostics ===");
    
    // Connection status
    Serial.printf("Status: %s\n", WiFi.status() == WL_CONNECTED ? "Connected" : "Disconnected");
    Serial.printf("SSID: %s\n", WiFi.SSID().c_str());
    Serial.printf("RSSI: %d dBm\n", WiFi.RSSI());
    
    // Network configuration
    Serial.printf("IP: %s\n", WiFi.localIP().toString().c_str());
